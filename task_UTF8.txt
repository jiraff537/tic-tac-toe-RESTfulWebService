Сделать REST-бекэнд для онлайн-игры крестики-нолики, стандартная версия 3x3.
Считаем, что играют 2 человека по сети. 
UI написан на JavaScript (писать UI не требуется), нужно просто реализовать обмен данными с сервером.
Должна быть возможность:
- зарегистрировать Участников 1 и 2
- сделать ход
- получить текущее состояние игрового поля
- получить текущий статус игры

Бекэнд должен быть реализован на Spring Boot, и собираться через Maven. В качестве сервера использовать Apache Tomcat.
Код должен сопровождаться набором unit-тестов.
Должен быть написан интеграционный тест, имитирующий игру 2 игроков.

####Поставка выполненного задания:
* исходный код (в git репозитории, например, github)
* приложение с readme(содержит инструкцию как запустить приложение)
* тестовых данных должно быть достаточно для проверки всех операций.
